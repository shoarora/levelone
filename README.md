# Level One

## Instructions
run `npm install` first time
run `npm start` to start

## Directory Structure
```
root
    - js              // our written js files
        - p5          // p5.js library
        - progress.js
    - node_modules    //stores node.js modules
    - index.html      // main.js renders this file
    - main.js         // the main file that runs
    - progress.html
```
